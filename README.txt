
# Billing Software Project

## Requirements
- Python 3.x
- pip install pillow mysql-connector-python
- MySQL Server (with user 'root' and password '')

## Files
- billing_app.py : The main GUI application using Tkinter
- logo.png : Logo used in the application
- setup_database.sql : Run this SQL script in MySQL to create the database and tables

## Running the App
1. Run the SQL script in your MySQL environment to create `billing_db`
2. Place `logo.png` and `billing_app.py` in the same folder
3. Run the app using:
   python billing_app.py
